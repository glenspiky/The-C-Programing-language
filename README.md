# Learning C Programming Language

This repository contains my notes, exercises, and small projects while learning the C programming language using *The C Programming Language* by Brian W. Kernighan and Dennis M. Ritchie (K&R).

## 📚 About This Repo

C is a powerful low-level programming language that forms the foundation of modern systems programming. This repo follows a structured learning path based on the classic C programming book.

## 🎯 Goals

- Understand the fundamentals of C programming
- Learn memory management and pointers
- Build strong problem-solving skills
- Complete exercises from each chapter of the book
- Write clean and efficient C code

## 🧠 Topics Covered

- Basic syntax and structure
- Variables and data types
- Control flow (if, switch, loops)
- Functions
- Arrays and strings
- Pointers and memory
- Structures
- File I/O
- Preprocessor directives
